# iOS Technical Test – Sample Implementation

This repository contains a sample implementation of the technical test from  
https://github.com/PaulLavoine/iOS_technical_test

## About

This project was created as a learning exercise and a first draft implementation of the test requirements.  
The goal was to explore architecture, implementation ideas, and improve iOS development skills.

It is **not intended to be a production-ready or fully optimized solution**.

## Purpose

- Practice iOS development concepts
- Experiment with architecture and implementation choices
- Learn and iterate through a real-world style technical exercise

## Notes

- The implementation focuses on clarity and learning rather than optimization
- Some parts may be simplified or incomplete
- Improvements and refactors are welcome
